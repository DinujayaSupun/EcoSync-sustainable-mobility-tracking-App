# ✅ HOW TO UPDATE YOUR .env FILE

# Replace the MONGO_URI line in server/.env with this format:
# Use the EXACT connection string from MongoDB Atlas (Step 2 above)

MONGO_URI=mongodb+srv://YOUR_USERNAME:YOUR_PASSWORD@cluster0.XXXXX.mongodb.net/sustainabilityDB?retryWrites=true&w=majority

# REPLACE:
# 1. YOUR_USERNAME → Your database username from Atlas (e.g., sustainabelDB)
# 2. YOUR_PASSWORD → Your database password 
# 3. XXXXX → Your cluster ID from the connection string Atlas gave you
#
# EXAMPLE:
# If Atlas gave you: mongodb+srv://user123:pass456@cluster0.abc12.mongodb.net/...
# Your XXXXX is: abc12
#
# Final should look like:
# MONGO_URI=mongodb+srv://sustainabelDB:MyP@ssw0rd@cluster0.pobsv1d.mongodb.net/sustainabilityDB?retryWrites=true&w=majority

# ⚠️ IMPORTANT:
# - No spaces before or after the =
# - Username and password are case-sensitive
# - The cluster ID (XXXXX) must match what Atlas gave you EXACTLY
# - If password has special characters like @, #, /, encode them:
#   @ = %40
#   # = %23
#   / = %2F
